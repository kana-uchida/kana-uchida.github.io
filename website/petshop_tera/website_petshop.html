PETSHOP・TERA

初めてペットを迎える方にも安心感が伝わる店舗サイト（シングルページ）

制作時間：3日

制作範囲：デザイン（ロゴ以外）/ コーディング

使用ツール：

Ps Photoshop Dw Dewamweaver

使用技術：HTML / CSS / jQuery / slick

コーディングのポイント

・レスポンシブ対応（PC/SP)

・スクロール時のフェードイン演出

・slickを使用したカルーセル実装

デザインの工夫

・優しいグリーンを基調に安心感を演出

・ペットの写真を主役にしたレイアウト

・カードUIで情報を整理

WEBサイトを見る

 

指示内容

※ 本作品は架空の店舗を想定したWebサイト制作課題です。

サイト名：PETSHOP・TERA

ディスクリプション： いっしょに暮らす、よろこびを。ワンちゃん・ねこちゃん・鳥さんが、あなたのお迎えを待っています！ 動物病院やペットホテル、トリミングなど家族に迎えた後もお客様をしっかりサポート。 初めての方でも安心です。

ナビゲーション： HOME / ペットショップ・テラの想い / NEWS / 当店人気ランキング

ロゴについて： 基本ロゴは変更不可です。ロゴによっては規定もあるので、その場合はしっかり規則に沿って使用してください。 今回の場合は、色の変更は不可です。 唯一可能なのは、背景に色が入っておりませんので、デザインに影響があるようでしたら、 ロゴと同じ大きさの白い楕円を配置するのはOKとします。

ターゲット：ファミリー層

イメージ：あたたかなイメージ／フレッシュなイメージ

事業内容： ショッピングモール内ペットショップ
・生体販売
・ペット用食品／ペット用雑貨の販売
・会員制度あり（しつけ教室予約優先／定期健診予約優先）

レスポンシブ対応：する

写真素材：指定のもの以外を使わないこと。
